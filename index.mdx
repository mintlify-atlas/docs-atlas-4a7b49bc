PycePlus builds on top of [Pygame-CE](https://pyga.me/) to give you a production-ready foundation for Python apps and games. Instead of writing your own main loop, event wiring, and resource monitoring from scratch, PycePlus provides them out of the box — so you can focus on your logic.

## What PycePlus gives you

<CardGroup cols={2}>
  <Card title="Structured mainloop" icon="rotate">
    A single `mainloop()` call handles rendering, events, FPS capping, per-second callbacks, and resource monitoring.
  </Card>
  <Card title="GUI framework" icon="window">
    Hierarchical `Frame` containers and bindable widgets with mouse events, padding, borders, and scroll support.
  </Card>
  <Card title="44 built-in events" icon="bell">
    System, network, database, file, memory, disk, and thread lifecycle events — all thread-safe.
  </Card>
  <Card title="Async downloads" icon="arrow-down-to-line">
    Resumable, pausable, cancelable file downloads with progress tracking, built on a managed thread pool.
  </Card>
  <Card title="File-system watcher" icon="eye">
    MD5-based change detection for files and directories, with `file.created`, `file.modified`, and `file.deleted` events.
  </Card>
  <Card title="Page navigation" icon="layer-group">
    Switch between named UI pages without destroying widgets — great for menus, settings screens, and multi-step flows.
  </Card>
</CardGroup>

## Get started in 3 steps

<Steps>
  <Step title="Install PycePlus">
    Clone the repository and install it alongside Pygame-CE.

    ```bash
    pip install pygame-ce
    git clone https://github.com/GrandKenzy/PycePlus.git
    cd PycePlus && pip install -r requirements.txt
    ```
  </Step>
  <Step title="Create a window and run the loop">
    ```python app.py
    import PycePlus as pp

    win = pp.window.new("My App", width=800, height=600, background_color=(30, 30, 30))

    pp.mainloop(FPS=60)
    ```
  </Step>
  <Step title="Add widgets and events">
    ```python app.py
    import PycePlus as pp

    win = pp.window.new("My App", width=800, height=600)

    frame = pp.Frame(width=200, height=100, background_color=(80, 40, 200), x=100, y=100)
    frame.bind("mouse.button.left.click", lambda w, pos: print("clicked!", pos))

    pp.mainloop(FPS=60)
    ```
  </Step>
</Steps>

<Note>
  PycePlus requires **Pygame-CE** (not the original Pygame). Make sure you install `pygame-ce` and not `pygame`.
</Note>
